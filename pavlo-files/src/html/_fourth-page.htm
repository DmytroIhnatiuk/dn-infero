What is Microsoft Excel?

Microsoft Excel is a leading spreadsheet application developed by Microsoft for Windows and Mac OS. Excel is used for storing, organising and manipulating data, and offers programming support via Microsoft Visual Basic for Applications (VBA).

Why chose to study Microsoft Excel with Infero Training?

Our standard Excel training consists of 3 levels each lasting a day: Introduction, Intermediate and Advanced.

If that is not enough, then choose from one of the addtional 7 courses Advanced Plus, Visual Basic 1 day, Visual Basic 3 days, PowerPivot, Pivot Tables, Dashboards, and Transition from earlier versions. If required we can create a bespoke course to match your requirements and specifications.

Each hands-on, intensive course is delivered as a combination of trainer demonstration, exercises to consolidate learner’s skills, an interactive support as required. All delegates are awarded with a certificate upon completion of the course at the end of the day.

Each course is suitable for anybody wanting to learn 365, 2019, 2016, 2013, 2010 or 2007, as it covers all the aspects and differences between the versions.
